About

xosd_message is a small tool to display messages on your X11 screen. just start it with xosd_message "Message you want to display". If you want to diplay more than a single word, just put the phrase in quotes as you see above. Thats it.

Install

Just call ./build and pray :) Make sure you have xosd and it's devel packages installed.
